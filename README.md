# ERRO - 2D横スクロールアドベンチャーゲーム (ソースコード)



##  (Overview)
本作は、台湾の大型展示会「新一代設計展（YODEX）」に出展した2D横スクロールアドベンチャーゲームです。
ゲームの企画、グラフィックデザイン（美術）、プログラミング（実装）の すべてを1人で開発（完全個人制作）しました。

本リポジトリは、就職活動における**プログラミングスキルのアピールを目的**としており、プロジェクト内のソースコード（C#スクリプト）のみを公開しています。

* **ジャンル:** 2D横スクロールアドベンチャー
* **開発環境:** Unity  / C#
* **制作期間:** 2023年9月 〜 2024年5月（約8ヶ月）
* **開発人数:** 1名（完全個人制作）
* **プレイ動画:** https://youtu.be/0M3mhPfIxQY
---

## (Technical Highlights)

### 1. コンポーネント指向によるキャラクター制御のモジュール化
* **概要:** プレイヤーの各アクション（攻撃、ダッシュ、射撃など）を単一のスクリプトに記述するのではなく、機能ごとに独立したコンポーネントとして分割・実装しました。
* **工夫した点:** 完全個人制作であるため、開発進行に伴うコードの肥大化を防ぐ必要がありました。ベースとなる移動処理から各アクションのロジックを切り離すことで、バグの特定を容易にし、機能追加やパラメータ調整がしやすい管理性の高いアーキテクチャを実現しました。
* **該当スクリプト:** `【PlayerController.cs】`, `【PlayerAttack.cs】`, `【PlayerDash.cs】`

### 2. 多彩なアクション派生と拡張性の高いレベルギミックの実装
* **概要:** 複数の攻撃モーションを組み合わせたコンボシステムの実装に加え、アイテム取得、地形ダメージ、ワープポータル、複数ルートに分岐するステージ進行など、アドベンチャーゲームとしての多様な要素を実装しました。
* **工夫した点:** 種類が豊富なステージギミックを効率よく管理する、プレイヤー側の判定処理コードを複雑化させることなく、新しいギミックや罠を容易に追加できる拡張性の高い設計を実現しています。また、攻撃の派生処理もステートやアニメーションイベントと連携させ、スムーズな操作感を持たせています。
